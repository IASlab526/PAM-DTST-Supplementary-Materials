# File Description

## `appendix/APPENDIX.pdf`

Supplementary appendix containing the experimental conditions based on the Taguchi L9 orthogonal array. The table includes 108 trials with the following fields:

- Run number
- Actuation configuration
- Excitation frequency, `f` in Hz
- Tendon pretension, `T_pre` in N
- PAM pressure, `P` in psi

## `videos/video_01_four_configuration_comparison.mp4`

Supplementary experimental video comparing four actuation configurations:

- NO PAM
- SEA
- PEA
- SPEA

The video shows representative cyclic motion and actuation behavior under the parameter conditions displayed in the video frame.
